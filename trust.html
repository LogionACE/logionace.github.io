Technical controls

How we protect evaluation data.

TLS 1.3

All data encrypted in transit using TLS 1.3.

AES-256

Evaluation data encrypted at rest with AES-256.

90-day purge

Evaluation logs and test artifacts automatically deleted after 90 days.

API-only

All testing via API or sandboxed environments. No production data access required.

Evaluation independence

How we ensure objectivity.

Cross-family judging

Subject models are never judged by their own model family. An OpenAI model is not judged by another OpenAI model.

Grounded evaluation

Obligation cases are scored against an independently maintained regulatory corpus, not the judge model's memory.

Stability protocol

Critical-severity cases are run three times. The case passes only if all three runs pass (worst-of-3).
