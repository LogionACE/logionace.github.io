Technical controls

How we protect evaluation data.

TLS 1.3

All data encrypted in transit using TLS 1.3.

AES-256

Evaluation data encrypted at rest with AES-256.

90-day purge

Evaluation logs and test artifacts automatically deleted after 90 days.

API-only

All testing via API or sandboxed environments. No production data access required.

Evaluation independence

How we ensure objectivity.

Proprietary evaluation engine

All evaluations are conducted through a purpose-built testing environment with deterministic checks, regulatory corpus grounding, and multi-pass verification.

Grounded evaluation

Obligation cases are scored against an independently maintained regulatory corpus, not model memory.

Stability protocol

Critical-severity cases are run three times. The case passes only if all three runs pass (worst-of-3).

Conflict of interest policy

How we protect evaluation integrity.

Fee-outcome firewall

Evaluation fees are fixed regardless of outcome. Payment structure does not change based on scores or verdicts. No pay-for-grade arrangements are accepted.

Affiliated system disclosure

If any system affiliated with LogionOS, Inc. is evaluated under the ACE protocol, the affiliation is publicly disclosed in the report and on the leaderboard.

Pre-release isolation

Evaluation results are not shared with evaluated parties before report finalization. Scores are determined by the automated pipeline and evidence review, not negotiation.
